## 三、智能投放策略优化
### （一）动态预算算法

预算基数 = 28日ROAS均值 × 目标CPA
调节因子 = (竞争指数/信誉评分) × 0.58
最终预算 = 基数 × (1 + 调节因子) × 时段系数
